Selectores

Los selectores vistos en el juego son:

  1. Selector de tipo (A)

    Selecciona todos los elementos de tipo A. El tipo se refiere al tipo de etiqueta, por lo que div, p y ul son tipos de elementos diferentes.

  2. ID Selector (#id)

    Selecciona el elemento con un id específico. Se puede combinar con el selector de tipo.

  3. Selector de descendientes (A B)

    Selecciona todo elemento B dentro de A. B se llama descendiente porque está dentro de otro elemento.

  4. Combinar selectores de descendiente e ID (#id A)

    Se puede combinar cualquier selector con el selector descendiente.

  5. Selector de clase (.nombre_de_clase)

    El selector de clase selecciona todos los elementos con ese atributo de clase. Los elementos solo pueden tener un id pero muchas clases.

  6. Combinar el selector de clase (A.nombre_de_clase)

    Puede combinar el selector de clase con otros selectores, como el selector de tipo.

  7. Combinar selectores (A, B)

    Gracias a la tecnología de Shatner, se seleccionan todos los elementos A y B. Se puede combinar cualquier selector de esta manera y se pueden especificar más de dos.

  8. Selector universal (*)

    Se puede seleccionar todos los elementos del documento con el selector universal.

  9. Combinar el selector universal (A*)

    Selecciona todos los elementos dentro de los elementos A

  10. Selector de hermanos adyacentes (A + B)

    De este modo, se seleccionan todos los B elementos que siguen directamente al elemento A. Los elementos que se suceden se denominan hermanos. Están en el mismo nivel, o profundidad.

    En el marcado HTML de este nivel, los elementos que tienen la misma sangría son elementos del mismo nivel.

  11. Selector general de hermanos (A ~ B)

    Puedes seleccionar todos los elementos del mismo nivel de un elemento que le siguen. Esto es como el selector adyacente, excepto que obtiene todos los siguientes elementos en lugar de uno. En el ejemplo, se seleccionan todos los elementos B que siguen a A

  12. Selector de niños (A > B)

    Puede seleccionar elementos que sean elementos secundarios directos de otros elementos. Un elemento secundario es cualquier elemento que está anidado directamente en otro elemento.

    Los elementos que están anidados más profundamente que eso se denominan elementos descendientes. En el ejemplo, se seleccionan todos los elementos B que son hijos directos de A.

  13. Primer pseudoselector secundario (:fist-child)

    Puede seleccionar el primer elemento secundario. Un elemento secundario es cualquier elemento que está directamente anidado en otro elemento. Puede combinar este pseudoselector con otros selectores.

  14. pseudoselector único secundario (:only-child)

    Puede seleccionar cualquier elemento que sea el único elemento dentro de otro. span:only-child selecciona los elementos span que son el único elemento secundario de algún otro elemento.

  15. Último pseudoselector secundario (:last-child)

    Puede seleccionar cualquier elemento que sea el único elemento dentro de otro.

    span:last-child selecciona todos los elementos span de last-child.

  16. N-ésimo pseudoselector secundario (:nth-child(n))

    Selecciona el enénesimo (1º, 2º...) elemento secundario en otro elemento.

    :nth-child(8) selecciona cada elemento que es el 8º hijo de otro elemento.

  17. Enésimo último selector de hijos (:nth-last-child(n))

    Selecciona los elementos secundarios de la parte inferior del elemento primario. Esto es como el enésimo hijo pero contando desde atrás.

  18. Primero del selector de tipo (:first-of-type)

    Seleeciona el primer elemento de ese tipo dentro de otro elemento.

  19. N-ésimo de selector de tipo (:nth-of-type(n))

    Selecciona un elemento específico en función de su tipo y orden en otro elemento, o en instancias pares (even) o impares (odd) de ese elemento.

  20. Selector de tipo N-ésimo con fórmula (:nth-of-type(An+B))

    La fórmula n-ésima de tipo selecciona cada enésimo elemento, iniciando el recuento en una instancia específica de ese elemento.

  21. Solo de selector de tipo (:only-of-type)

    Selecciona el único elemento de su tipo dentro de otro elemento.

  22. Último del selector de tipo (:last-of-type)

    Selecciona hasta el último elemento de ese tipo dentro de otro elemento. Recuerde que type se refiere al tipo de etiqueta, por lo que p y envergadura son de diferentes tipos.

  23. Selector vacío (:empty)

    Selecciona elementos que no tienen ningún otro elemento dentro de ellos.

  24. Pseudoclase de negación (:not(X))

    Puede usar esto para seleccionar todos los elementos que no coincidan con el selector "X".

  25. Selector de atributo ([attribute])

    Seleccione todos los elementos que tienen un atributo específico

  26. Selector de atributos (A[attribute])

    Combine el selector de atributos con otro selector (como el selector de nombre de etiqueta) agregándolo al final.

  27. Selector de valor de atributo ([attribute="value"])

    Los selectores de atributos distinguen entre mayúsculas y minúsculas, cada carácter debe coincidir exactamente.

  28. El atributo comienza con el selector ([attribute^="value"])

    Seleccione todos los elementos con un valor de atributo que comience con caracteres específicos

  29. El atributo termina con selector ([attribute$="value"])

    Seleccione todos los elementos con un valor de atributo que termine con caracteres específicos

  30. Selector de comodines de atributo (([attribute*="value"]))

    Seleccione todos los elementos con un valor de atributo que contenga caracteres específicos en cualquier lugar. Los selectores de atributos distinguen entre mayúsculas y minúsculas, cada carácter debe coincidir exactamente.
